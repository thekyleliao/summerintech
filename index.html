SummerInTech(What was to be!)

We aimed to make a website that had many components. We wanted to run an apache web server, a flask backend, mysqldatabase, all via a command line interface. We failed.

Our Original Site Idea

SummerinTech was meant to be a website that enabled you to view webscraped data from our database about student opportunities, hence the name summer in tech. We managed to create one formula for web scraping.

This is the culmination of our work in reality. { "NYCDOE Minecraft Net Zero Challenge": { "Deadline": "May 27, 2022", "Event": "April 11-May 27, 202" }, "Acing Your Resume and Interview Questions": { "Deadline": "June 7, 2022", "Event": "June 7, 2022" }, "YVote ChangeMakers Summer Institute\u00a0": { "Deadline": "June 6, 2022", "Event": "July 12-Aug 16, 2022" }, "Literacy Unbound Summer Institute": { "Deadline": "June 3, 2022", "Event": "July 18 - 30, 2022" }, "First Tech Fund 2022-2023 Fellowship": { "Deadline": "June 1, 2022", "Event": "Ongoing" }, "See MCC Theater's Production of Soft": { "Deadline": "June 19, 2022", "Event": "May 12-June 19, 2022" }, "ONE Lab Summer Studio 2022": { "Deadline": "Ongoing ", "Event": "June 20-July 28, 202" }, "Next Gen Democracy Camp": { "Deadline": "May 27, 2022", "Event": "June 27- July 1, 202" } } We made a JSON file that is webscraped and organized from the NYC student opportunities page. Our goal was to make web scraping algos for all sorts of city sites from fire department to the cities commission for animal rights to help students find the opportunity for them. A lot of these sites have terrible SEO and purposely obscure the chance to work for them. It’s a you gotta know something thing for a lot of the postings. We wanted to eliminate that by making one site dedicated to a very specific set of opportunities that wouldn’t be posted on linkedin or job finding sights. We did not succeed. We took on too much. All those things we wanted to use? we have never used them before. As a result, we have made but one thing, an apache web server hosting this static asset. A mildly depressing lession that you really should not try to integrate like 5 different technologies in 24 hours. The rest of our story will go on devpost. -kyle.
